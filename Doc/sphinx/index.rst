Architecture Overview
=====================

The AI Scientist consists of four primary components:

1. **Idea Generation**: Generates multiple novel research hypotheses with thorough literature novelty assessment
2. **Experimental Iteration**: Executes code-based experiments in secure Dockerized environments  
3. **Paper Write-up**: Composes academic papers with proper LaTeX formatting and citations
4. **Automated Peer Review**: Performs rigorous evaluation using ICLR-style criteria with iterative refinement
